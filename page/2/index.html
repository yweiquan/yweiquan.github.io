我们知道，程序运行时，允许改变程序结构或变量类型，这种语言称为动态语言。从这个观点看，Perl，Python，Ruby是动态语言，C++，Java，C#不是动态语言。

AsyncTask使用须知

Posted on 2014-01-06   |  

版本变化

在1.6（Donut）之前：

从Android API 3（1.5 Cupcake）开始，AsyncTask被引入用来帮助开发者更简单地管理线程。在第一版的AsyncTask，任务是串行调度。一个任务执行完成另一个才能执行。由于串行执行任务，使用多个AsyncTask可能会带来有些问题。所以这并不是一个很好的处理异步（尤其是需要将结果作用于UI试图）操作的方法。

从1.6到2.3（Gingerbread）：

后来Android团队决定让AsyncTask并行来解决1.6之前引起的问题，这个问题是解决了，新的问题又出现了。很多开发者实际上依赖于顺序执行的行为。于是很多并发的问题蜂拥而至。

Android Loader 机制

Posted on 2013-05-12   |  

应用场景

先说一个典型的应用场景，打开某个界面时，需要先加载本地缓存的数据，然后再请求网络获取最新的数据，刷新UI。这个时候就可以使用Loader来简化操作。
或者应用经常性的对数据源进行增删改，需要UI及时作出相应的刷新时，Loader也适用。

Loader特性

Loader机制从Android3.0和Fragment同时引进。它使得在Activity或Fragment中异步加载数据变得简单。Loader具有如下特性：

  • 它们对每个Activity和Fragment都有效。
  • 他们提供了异步加载数据的能力。
  • 它们监视数据源的一将一动并在内容改变时传送新的结果。
  • 当由于配置改变而被重新创建后，它们自动重连到上一个加载器的游标，所以不必重新查询数据。

作者简介

Paul Graham 保罗•格雷厄姆，硅谷创业之父。1964年出生于匹兹堡，在康奈尔大学读完本科，然后在哈佛大学获得计算机科学博士学位。1995年，他创办了Viaweb，帮助个人用户在网上开店。1998年夏天，Yahoo!公司收购了Viaweb，收购价约为5000万美元。2005年，创建风险投资公司YCombinator，将自己的理论转化为实践，目前已经资助了80多家创业公司。

内容简介

黑客，这里指优秀程序员。主要介绍黑客的爱好和动机，讨论黑客成长、黑客对世界的贡献以及编程语言和黑客工作方法等所有对计算机时代感兴趣的人的一些话题。书中的内容不但有助于了解计算机编程的本质、互联网行业的规则，还会帮助读者了解我们这个时代，迫使读者独立思考。
